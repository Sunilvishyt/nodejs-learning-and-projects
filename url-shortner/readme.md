# URL Shortener Project

This project is a URL shortener built with Node.js and Express. It allows users to shorten long URLs and provides a convenient way to share links.

## Features

- Shorten long URLs
- Redirect to the original URL
- Display a list of shortened URLs

## Installation

1. Clone the repository: `https://github.com/Sunilvishyt/nodejs-learning-and-projects`
2. go the the url shortner directory
3. Install the dependencies: `npm install`
4. Start the server: `npm start`

## Usage

1. Open your web browser and go to `http://localhost:8000`
2. Enter the URL you want to shorten in the form.
3. Click the "Generate" button to create a shortened URL.
4. Click on the shortened URL to be redirected to the original URL.
5. view the list of shortened URLs, under the "short urls" heading.
